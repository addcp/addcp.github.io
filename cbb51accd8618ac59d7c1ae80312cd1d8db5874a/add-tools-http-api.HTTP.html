add-tools-http-api. HTTP

O Serviço de HTTP é usado para fazer solicitações HTTP e lidar com erros e respostas de uma maneira específica.

Methods

created()

created é um dos métodos do ciclo de vida do serviço no Moleculer. Ele é chamado uma vez, quando o serviço é criado.
Aqui, ele está configurando um cliente HTTP personalizado para o serviço, com opções para registro e formatação de resposta. O cliente é então armazenado no serviço para uso posterior.
